安装

Debian / Ubuntu:

1
2

apt-get install python-pip
pip install shadowsocks

CentOS:

1
2

yum install python-setuptools && easy_install pip
pip install shadowsocks

参数说明

参数 说明
server 监听地址
server_port 监听端口
password 客户端连接密码
timeout 超时时间
method 加密方法，可选aes-128-cfb, aes-192-cfb, aes-256-cfb, bf-cfb, cast5-cfb, des-cfb, rc4-md5, chacha20, salsa20, rc4, table
fast_open 是否启用TCP-Fast-Open

启动服务

1

$ ssserver -c /etc/shadowsocks.json --log-file /var/log/shadowsocks.log -d start
